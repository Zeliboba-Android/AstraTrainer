Lesson 1: Navigating the File System

Задание: Перейдите в директорию "Documents", создайте там новую директорию "MyFolder" и выведите список содержимого директории.

Подсказка: Используйте команду `cd Documents`, чтобы перейти в директорию "Documents". Затем используйте `mkdir MyFolder`, чтобы создать новую директорию, и `ls`, чтобы вывести список содержимого.
